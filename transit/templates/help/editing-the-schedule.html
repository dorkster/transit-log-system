Now that Drivers, Vehicles, and Trip Types and Clients, Destinations, and Templates have been set up, it's time to start editing schedules.

The Edit Schedule page can be accessed via the main menu under Schedule >> Edit Schedule.

(From left to right)
  • Print the schedule or the daily reports
  • Select the date
  • Switch to the drivers' View mode
Create a new shift
Edit or delete a shift
(From left to right)
  • Create a new trip or activity
  • Select an available template to insert its contents into the schedule
Edit or delete the trip or activity
Trips and activities can be canceled from here
For trips, there are shortcuts to copy the trip, create a return trip, or create a client
Change the order of trips
See additional information about a trip
Clicking "Show extra columns" will also reveal this information
Drivers and vehicles can be easily assigned by dispatchers

There are two types of entries that can be added to the schedule: activities and trips.

Activities are suitable for things such as staff meetings, designated office time, etc. They consist of a Description, plus an optional Start Time and End Time.

Trips are the primary focus of the schedule, describing a person's ride from one Address to a Destination. The information in a Trip can be divided into three parts: Client Information, Trip Information, and Log Information.

Mark a trip as canceled
The name field has a dropdown with a list of all the available Clients. Selecting one will fill in all the available information for that Client.
If the name entered isn't in the Client database, a warning will be displayed and the option to add them to the database will be enabled.
Fill in the phone number from the Client information
Times are entered in standard 12-hour format. Time entry is forgiving and aware of work hours, so simply entering "130" would convert to "1:30 PM".
Fill in the home address from the Client information
Fill in the phone number if the address matches information in the Destination database
The address fields have a dropdown with a list of all the available Destinations. Selecting one will fill in the phone number if applicable.
Tags are user-defined labels that can be applied to Trips. Each tag is separated by a comma ( , ).
Clicking on one of the frequent tags will add it to this Trip's list of tags.
While most of the Log Information will be filled in by the drivers, the assigned Driver and Vehicle for the Trip can be entered here.
(From left to right)
  • Create a return trip: When enabled, clicking Save Changes will also create a new Trip with the pick-up and destination swapped.
  • Add to Clients: If the name entered doesn't match anyone in the Client database, enabling this option will create a new Client entry when clicking Save Changes.
