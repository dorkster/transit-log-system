Once a schedule has been created, drivers can then make use of it. The schedule can continue to be modified and updated by other staff, as any changes will automatically be reflected in the driver's view.

The View Schedule page can be accessed via the main menu under Schedule >> View Schedule.

The typical workflow for drivers looks something like this:

  1. Fill out the Pre-Trip form for their current Shift.
  2. Start the Shift. The full odometer mileage and the current time are entered.
  3. Pick Up and Drop Off Trip passengers until finished. Mileage, time, and collected money are recorded.
  4. (Optional) Enter Fuel information if applicable.
  5. End the Shift. Once again the full odometer mileage and current time are entered.
Pre-Trip items can be marked as passing or failing
If an item is marked as failing, a vehicle issue can be filed here. More information about vehicle issues can be found in the next section, Vehicle Status.
