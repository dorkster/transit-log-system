Reports give staff a way to view statistical data about all the activity over a given time period.

The Report page for the most recent month can be accessed via the main menu under Reports >> View this month's report.

(From left to right)
  • Downloads the entire report as an Excel (XLSX) file. Please be aware that the file contains only the output data and no formulas.
  • Select the date. The arrows jump between entire months, but more granular control of the date range is available when clicking on the calendar.
Detected errors, such as incomplete log data, are displayed here.
Reports include a summary of usage for all vehicles, as well as a summary for each individual vehicle. Also included are summaries of elderly/ambulatory riders, collected money, and individual driver activity.
Click on a date to go to the schedule where this vehicle was used.
