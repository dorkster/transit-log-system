Before using the schedule system, it is important to first set up the available drivers and vehicles, as well as the different trip types.

Your administrator has likely already done this for you, but it is worth reviewing this section to learn how to add, edit, and re-order these items.

Drivers

The Drivers page can be accessed via the main menu under Schedule >> Drivers.

Create a new driver
Edit an existing driver
Change the order that drivers will appear

When creating or editing a driver, the following properties can be changed:

  • Name - The name of the driver
  • Color - The background color used for the driver on the schedule
  • Is logged? - Whether or not this driver is displayed in various places. Generally, employed drivers should be logged, and volunteers should not be logged.

Vehicles

The Vehicles page can be accessed via the main menu under Vehicles >> Vehicles.

Everything here is similar to dealing with drivers, except vehicles only have two properties: Name and Is logged?

In most scenarios, all vehicles will be logged. The notable exception being a "personal" vehicle.

Trip Types

The Trip Types page can be accessed via the main menu under Schedule >> Trip Types.

For monthly reports that require a breakdown of trips by type, said types can be set up here. Again, the interface here is identical to editing drivers and vehicles. Trip types only have one property, which is their display Name.
