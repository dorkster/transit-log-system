It is highly recommended to set up clients and templates before creating schedules. Doing so will make creating recurring trips easier and less prone to typo errors.

Clients

The Clients page can be accessed via the main menu under Schedule >> Clients.

The clients created here will appear in the drop-down of the Name field when creating trips in a template or schedule. Defining clients here allows fields such as the address and phone number to be automatically filled in when creating trips.

Filter the available clients by elderly and ambulatory status...
...Or simply search by name and address
Edit or delete an existing client

Clicking "Show extra columns" will also reveal this information

The available fields when creating a client are fairly straight-forward:

  • Name - The client's full name
  • Address - The client's home address
  • Phone - The client's home and/or cell phone number
  • Elderly - Is the client elderly?
  • Ambulatory - Is the client ambulatory?
  • Tags - A list of user-defined labels, separated by commas.

Destinations

The Destinations page can be accessed via the main menu under Schedule >> Destinations.

Much like clients, destinations provide a way to quickly populate fields when creating trips in a template or schedule. When creating a trip, destinations defined here will appear in the drop-down of the Pick-Up Address and Destination Address fields.

Destinations only have two fields to define: the Address and, optionally, the Phone number.

Templates

The Templates page can be accessed via the main menu under Schedule >> Templates.

For things such as daily schedules and regular group outings, templates can be created to speed up inserting multiple trips into the schedule.
