It is highly recommended to set up clients and templates before creating schedules. Doing so will make creating recurring trips easier and less prone to typo errors.

Clients

The Clients page can be accessed via the main menu under Schedule >> Edit Clients.

The clients created here will appear in the drop-down of the Name field when creating trips in a template or schedule.

The available fields when creating a client are fairly straight-forward:

  • Name - The client’s full name
  • Address - The client’s home address
  • Phone - The client’s home and/or cell phone number
  • Elderly - Is the client elderly?
  • Ambulatory - Is the client ambulatory?
  • Tags - A list of user-defined labels, separated by commas.
