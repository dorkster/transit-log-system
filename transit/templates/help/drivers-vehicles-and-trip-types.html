Before using the schedule system, it is important to first set up the available drivers and vehicles, as well as the different trip types.

Your administrator has likely already done this for you, but it is worth reviewing this section to learn how to add, edit, and re-order these items.

Drivers

The Drivers page can be accessed via the main menu under Schedule >> Drivers.

Create a new driver
Edit an existing driver
Change the order that drivers will appear

When creating or editing a driver, the following properties can be changed:

  • Name - The name of the driver
  • Color - The background color used for the driver on the schedule
  • Is active? - Determines if the driver will be shown in various places. The most common usage is to flag former (or current) employees that no longer drive for your team.
  • Include in general reports? - Log data from trips by this driver will be included on reports. Also determines if the driver can be selected in certain places, such as when filing a vehicle issue. Generally, employed drivers should be included, while volunteers should not.

Vehicles

The Vehicles page can be accessed via the main menu under Vehicles >> Vehicles.

When creating or editing a vehicle, the following properties can be changed:

  • Name - The name of the vehicle
  • Is active? - Determines if the vehicle will be shown in various places. The most common usage is to flag decommissioned vehicles.
  • Report mileage/service hours? - Log data for trips by with this vehicle will be included entirely on reports. When turned off, this data won't be applied to certain totals. It may make sense to turn this off, for example, on a staff member's personal vehicle.

Trip Types

The Trip Types page can be accessed via the main menu under Schedule >> Trip Types.

For monthly reports that require a breakdown of trips by type, said types can be set up here. Again, the interface here is identical to editing drivers and vehicles. Trip types have the following properties:

  • Name - The name of the trip type
  • Report trip counts? - Determines if trips flagged with this type are counted in the relevant sections of reports.
